## Optics Solidity

On-chain implementations of Optics in Solidity.

### Setup

- `npm install --dev`
- `brew install jq` &nbsp; OR &nbsp; `sudo apt-get install jq`
- `npm install -g ethlint`, check it is installed using `solium -V`

### Build

- `npm run compile`
- `npm run test`
